# Twilight – A Pixel-Art Walking Experience

A cozy, atmospheric pixel-art side-scroller where you walk along a twilight road under changing moons, whistling to seagulls, dodging puddles, and discovering small secrets in a quiet, melancholic world.

## Controls

### Keyboard (Desktop)
- **A / ←** – Walk left  
- **D / →** – Walk right  
- **Shift + A/D** – Run (hold)  
- **Q** – Change outfit  
- **W** – Whistle (calls seagulls, triggers events)  
- **S** – Shooting star  
- **R** – Toggle rain  
- **M / N** – Previous / Next moon phase  

### Mobile / Touch
On-screen retro gamepad appears automatically:  
- Left side: movement + outfit/whistle buttons  
- Right side: moon controls + primary actions (shooting star, rain)  
Double-tap movement buttons to run

## Features
- Dynamic moon phases with unique lighting & reflections  
- Weather (rain with audio), puddles with reflections  
- Whistling → seagull interaction & shell collection  
- Ambient particles, lamp glows, wire birds, distant silhouettes  
- Title screen, subtle secrets (e.g. moon expressions)  
- Deterministic world generation (same seed every time)

## Tech Stack
- Pure vanilla JavaScript + HTML5 Canvas  
- No external libraries or frameworks  
- Seeded random number generator for consistent world  
- Touch-optimized mobile controls with haptic feedback  
- Procedural audio (rain pink noise + drops, whistle tones)

## How to Run Locally
1. Clone or download this repository
2. Open `index.html` in a modern browser (Chrome, Firefox, Edge, Safari)
3. Enjoy!
